# Mistake Avoidance Measurement v1

## Purpose

Measure whether Moon helps prevent repeated mistakes.

## Mistakes To Test

Moon should help prevent:

- repeating rejected decisions
- missing known constraints
- ignoring prior rationale
- reopening settled issues without new evidence

## Study Method

Give participants tasks with tempting wrong paths.

Examples:

- propose adding ingestion before trust boundaries are stable
- expand Decision Memory before boundary reliability improves
- claim economic value from TTC signal alone
- treat reliability evidence as economic evidence
- build an agent before constitutional constraints are validated

## Captured Measures

- whether participant chooses rejected path
- whether participant identifies the relevant constraint
- whether participant cites evidence
- whether participant explains why the path is blocked
- time to detect the mistake

## Economic Interpretation

Avoiding one serious repeated mistake can be more valuable than saving a small amount of reading time.
