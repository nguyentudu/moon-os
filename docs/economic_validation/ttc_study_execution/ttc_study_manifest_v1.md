## Boundaries

This study execution layer does not:

- claim PMF
- claim onboarding reduction
- claim economic value
- modify Decision Definition
- expand Decision Memory
- create fake participants
- create fake measurements
- create synthetic study results
- run automatic scoring without preserving raw data

## Evidence Principle

Human evidence first.

Raw answers and timings must be preserved before analysis.
