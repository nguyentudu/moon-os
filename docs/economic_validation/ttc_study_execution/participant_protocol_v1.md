# Participant Protocol v1

## Purpose

Define the participant workflow for the Moon TTC Study.

## Participant Steps

1. Open the TTC Study UI.
2. Enter Participant ID.
3. Select assigned mode:
   - Manual Context Discovery
   - Decision Memory Assisted Context Discovery
4. Consent to raw answer capture.
5. Start the study.
6. Answer each context question.
7. Copy or download raw evidence at completion.
8. Send raw evidence back manually.

## Participant Rules

- Answer in your own words.
- Do not switch mode during the study.
- Do not use outside AI assistance unless the study coordinator explicitly assigns that condition.
- Manual mode should use repository/context reading.
- Decision Memory mode should use the provided Decision Memory context.

## Completion Message

Study complete.

Thank you.

Results will be analyzed separately.

No metrics.

No claims.
