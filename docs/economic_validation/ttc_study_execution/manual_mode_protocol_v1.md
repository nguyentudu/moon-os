# Manual Mode Protocol v1

## Purpose

Define Manual Context Discovery for the TTC Study.

## Participant Instruction

Use the Moon repository and available docs to answer each question.

Do not use the Decision Memory assisted context provided in the assisted mode.

## Captured Data

For each question:

- participant_id
- mode
- question_id
- question
- answer
- start_time
- end_time
- elapsed_seconds
- created_at

## Boundary

Manual mode is not scored in the UI.

Manual mode results are raw evidence only.
