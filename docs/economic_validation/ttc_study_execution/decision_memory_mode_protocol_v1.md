# Decision Memory Mode Protocol v1

## Purpose

Define Moon Decision Memory Assisted Context Discovery for the TTC Study.

## Participant Instruction

Use the Decision Memory context shown in the UI to answer each question in your own words.

The context may include:

- retrieved records
- reason
- evidence links
- current validity
- unknowns

## Captured Data

For each question:

- participant_id
- mode
- question_id
- question
- answer
- start_time
- end_time
- elapsed_seconds
- created_at

## Boundary

Decision Memory mode is not scored in the UI.

No winner is declared.

No improvement claim is shown.
