## Validation Notes

The UI changes display copy only. The TTC protocol remains unchanged:

- participant selects assigned mode
- participant answers the same question set
- timing is captured per answer
- raw evidence is copied or downloaded manually
- results are analyzed separately

## Known Limitations

This limitation should remain visible until a localization QA pass is completed.
