# Translation Policy v1

## Purpose

Translations must reduce language bias without changing what the study measures.

## Translation Rules

Translations must:

- preserve research meaning
- preserve question intent
- preserve study neutrality
- avoid leading wording
- avoid persuasive wording
- avoid simplifying away important constraints
- preserve the difference between Manual Mode and Decision Memory Mode

## Participant Answers

Participant answers must not be translated automatically.

If later analysis needs translation, the translation must be treated as a separate analysis artifact and the original answer must remain preserved.

## Evidence and Context

Evidence links, record titles, and decision record identifiers remain unchanged across languages so that both language versions point to the same source material.

## Claims Boundary

Localization does not create:

- TTC improvement evidence
- economic evidence
- product-market fit evidence
- onboarding reduction evidence

It only improves participant accessibility and study validity.
