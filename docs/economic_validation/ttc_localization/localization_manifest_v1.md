# TTC Localization Manifest v1

## Purpose

TTC Localization allows participants to complete the Moon Time To Context study in a supported language without changing the study protocol.

The goal is to reduce language bias in Time To Context measurement by making instructions, consent copy, questions, and completion copy understandable in English or Vietnamese.

## Scope

This layer applies to:

- TTC Study landing page copy
- consent copy
- question text
- manual mode instructions
- Decision Memory context display
- completion screen copy
- raw evidence language metadata

## Non-Goals

This layer does not:

- score answers
- translate participant answers
- modify participant answers
- change TTC metrics
- add analytics
- add dashboards
- make economic claims
- change the study protocol

## Study Validity Rule

Every localized version must preserve the same study intent, neutrality, and data capture fields.

The only new stored field is:

- `participant_language`
