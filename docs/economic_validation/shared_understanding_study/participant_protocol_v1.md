# Participant Protocol v1

## Instructions

You will receive several Moon OS reasoning cases.

For each case:

1. Retrieve relevant context from Moon docs or Decision Memory.
2. Explain what evidence matters.
3. Explain the reasoning chain.
4. Discuss tradeoffs and constraints.
5. Make a recommendation.
6. State uncertainty or missing evidence.

## Required Answer Fields

For each case, provide:

- answer summary
- evidence used
- rationale
- risks
- constraints
- recommendation
- uncertainty

## Prohibited Shortcuts

Do not answer by copying a single existing line.

Do not answer only with a document path.

Do not claim Moon should advance if evidence is insufficient.

## Participant Data Policy

No participant responses are created in this setup task.

Future responses must be preserved exactly as submitted.
