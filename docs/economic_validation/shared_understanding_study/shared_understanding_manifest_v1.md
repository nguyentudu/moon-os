# Shared Understanding Study Manifest v1

## Purpose

The Shared Understanding Study tests whether Moon Decision Memory helps participants reconstruct Moon reasoning and judgement independently.

This study moves beyond Time To Context:

Shared Retrieval

↓

Shared Reasoning

↓

Shared Understanding

## Core Question

Can different participants independently produce similar reasoning and judgement when the answer is not already available in the repository?

## Scope

The study evaluates:

- reasoning structure
- tradeoff awareness
- constraint awareness
- boundary awareness
- judgement quality
- independence

## Non-Goals

This study does not:

- modify Decision Memory
- modify the Decision Definition
- expand Decision Memory
- run a Time To Contribution pilot
- claim Team Continuity
- claim Shared Understanding confirmed
- claim economic value
- create participant answers
- build UI
- deploy Railway
