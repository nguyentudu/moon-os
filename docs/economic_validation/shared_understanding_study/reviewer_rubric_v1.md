# Reviewer Rubric v1

## Purpose

This rubric evaluates whether answers show retrieval, reasoning, judgement, or emerging shared understanding.

## 1. Reasoning Structure

Review:

- coherent chain?
- causal logic?
- evidence linked?

Labels:

- strong
- adequate
- weak

## 2. Tradeoff Awareness

Review:

- recognizes competing options?
- discusses risks?
- explains rejected paths?

Labels:

- strong
- adequate
- weak

## 3. Constraint Awareness

Review:

- respects project gates?
- respects validation requirements?
- does not skip required evidence?

Labels:

- strong
- adequate
- weak

## 4. Boundary Awareness

Review:

- avoids overclaiming?
- recognizes uncertainty?
- preserves current status?

Labels:

- strong
- adequate
- weak

## 5. Judgement Quality

Review:

- recommendation justified?
- recommendation consistent with evidence?
- recommendation appropriately cautious?

Labels:

- strong
- adequate
- weak

## 6. Independence

Review:

- answer copied?
- answer synthesized?
- answer demonstrates reasoning?

Labels:

- copied retrieval
- synthesized reasoning
- independent judgement

## Reviewer Comment

Reviewer comments are required for every case.
