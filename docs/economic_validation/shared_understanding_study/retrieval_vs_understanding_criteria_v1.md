# Retrieval vs Understanding Criteria v1

## SHARED_RETRIEVAL

Participant mainly repeats existing statements.

Indicators:

- answer closely mirrors known text
- answer includes source paths but little explanation
- recommendation is absent or copied
- no tradeoffs discussed
- no uncertainty named

## SHARED_REASONING

Participant reconstructs similar logic.

Indicators:

- evidence is connected into a causal chain
- participant explains why a gate exists
- participant recognizes what would change the conclusion
- answer is not merely a copied phrase

## SHARED_JUDGEMENT

Participant reaches similar conclusions through independently justified reasoning.

Indicators:

- participant compares options
- recommendation is justified
- risks are explicit
- constraints are respected
- uncertainty is visible

## SHARED_UNDERSTANDING

Participants consistently reconstruct reasoning and judgement across multiple novel cases.

Required evidence:

- multiple participants
- multiple novel cases
- consistent reasoning patterns
- consistent judgement patterns
- independent reviewer agreement
- no direct-answer dependency

## Current Claim Status

Current status:

SHARED_RETRIEVAL evidence exists.

SHARED_UNDERSTANDING is not confirmed.
