# Early TTC Signal Summary v1

## Summary

Moon has imported real TTC study evidence from six participants.

Decision Memory mode shows an early Time To Context reduction signal compared with manual context discovery.

This is not economic evidence.

## Key Numbers

| Metric | Manual | Decision Memory |
| --- | ---: | ---: |
| Participants | 3 | 3 |
| Answers | 18 | 18 |
| Average time per question | 11.21 minutes | 2.26 minutes |
| Median time per question | 11.85 minutes | 1.21 minutes |
| Average total run time | 67.27 minutes | 13.57 minutes |
| Median total run time | 65.82 minutes | 6.99 minutes |

## Early Signal

Claim level: EARLY_TTC_SIGNAL

The data suggests Decision Memory can reduce the time needed to answer Moon context questions.

The data does not prove:

- economic value
- Team Continuity
- PMF
- business readiness
- durable shared understanding

## From TTC to Continuity Loop

Time To Context currently measures:

Question -> Answer

Moon OS must eventually test:

Signal -> Why it matters -> One Action -> Response -> Next Signal

This is the bridge from Time To Context to Time To Contribution.

Do not implement this loop yet. Record it as the next research direction.

## Recommended Next Step

Run a blinded Answer Quality Review, then design a Time To Contribution pilot that tests whether participants can use the retrieved context to make a correct next action.
