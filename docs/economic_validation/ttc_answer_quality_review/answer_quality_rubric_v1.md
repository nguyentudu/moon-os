# Answer Quality Rubric v1

## Purpose

This rubric evaluates TTC answers separately from answer speed.

It does not create an economic claim, Team Continuity claim, Shared Understanding claim, or Time To Contribution claim.

## Categories

### Correctness

- factually correct: answer states the core reason accurately
- partially correct: answer captures part of the reason but misses an important qualifier
- incorrect: answer conflicts with the evidence

### Completeness

Review whether the answer includes:

- major elements
- rationale
- consequence or current implication

Labels:

- complete
- adequate
- incomplete

### Source Usage

Review whether:

- a source path is included
- the source appears appropriate
- the source supports the claim

Labels:

- strong
- adequate
- absent

### Reasoning Quality

Review whether:

- reasoning chain is present
- causal explanation is present
- conclusion is justified

Labels:

- strong
- adequate
- weak

### Boundary Awareness

Review whether the answer:

- acknowledges uncertainty
- avoids overclaiming
- reflects current project status

Labels:

- strong
- adequate
- weak

## Review Comment Requirement

Every scored answer must include a reviewer comment.

## Claim Boundary

Allowed:

- EARLY_QUALITY_SIGNAL

Not allowed:

- ECONOMIC_EVIDENCE
- SHARED_UNDERSTANDING_CONFIRMED
- TEAM_CONTINUITY_VALIDATED
- PMF
