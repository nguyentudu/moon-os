## Correctness Findings

Both modes produced mostly correct answers.

Decision Memory answers more consistently preserved qualifiers such as:

- unstable boundary evidence
- usable but insufficient evidence
- no premature value claims

## Completeness Findings

Manual answers often captured the main reason but omitted consequence or current status.

Decision Memory answers were generally adequate to complete, especially P-005 and P-006.

## Source Usage Findings

Decision Memory source usage was stronger overall because P-005 and P-006 frequently included source paths.

P-001 remained mostly unsourced.

Manual source usage was inconsistent.

## Reasoning Quality Findings

Decision Memory answers more often reconstructed the causal chain.

Manual answers often retrieved the right topic but compressed the reasoning.

## Boundary Awareness Findings

Decision Memory answers more often avoided overclaiming and preserved current-status language.

Manual answers were usually not overclaiming, but caution was less explicit.

## P-001 Review

P-001 is a quality and speed caveat.

P-001 was slower than P-005/P-006 and produced shorter answers with limited source usage.

P-001 does not invalidate the quality signal, but it prevents any claim that Decision Memory always produces fast, source-rich answers.

## Speed vs Quality Observations

Decision Memory was faster in the TTC time analysis and at least equal or higher quality in this internal blinded quality review.

This supports an early quality signal only.
