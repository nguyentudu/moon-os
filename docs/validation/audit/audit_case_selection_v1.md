# Audit Case Selection v1

## Purpose

Define the five scenarios selected for constitutional benchmark audit.

## Selected Cases

| Scenario ID | Scenario Name | Domain | Selection Reason |
| --- | --- | --- | --- |
| T1 | Old Sensitive Memory | Trust | Tests whether trust rejection is independently reasoned through consent and sensitivity. |
| C1 | Forgotten Commitment | Commitment | Tests high-concern continuity and priority escalation. |
| G1 | Goal Drift | Goal | Tests continuity reasoning from goal state and metric signals. |
| A1 | Attention Overload | Attention | Tests attention suppression and prioritization under overload. |
| T2 | Fresh Start Protection | Trust | Tests active boundary override and trust rejection. |

## Coverage

The selection covers:

- Trust
- Commitment
- Goal
- Attention
- Today visibility
- Task Writer artifact generation
- Suppression and rejection

## Non-Implementation Boundary

This selection does not create new scenarios or change benchmark outcomes.
