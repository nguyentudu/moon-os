# Benchmark Examples v1

## Purpose

These examples illustrate benchmark interpretation.

Examples only. No implementation.

## Example 1: Forgotten Commitment

Scenario:

Forgotten Commitment

Expected:

- HIGH_CONCERN
- PRIORITIZE
- ALLOW

Actual:

- CONCERN
- SURFACE
- ALLOW

Result:

BORDERLINE

Reason:

Severity lower than expected.

Variance:

Continuity Variance and Attention Variance.

Severity:

Moderate.

Root Cause:

Commitment urgency was underweighted.

Resolution Candidate:

Review commitment risk and timing-fit interpretation.

## Example 2: Old Sensitive Memory

Scenario:

Old Sensitive Memory

Expected:

REJECT

Actual:

ALLOW_WITH_CAUTION

Result:

FAIL

Severity:

CRITICAL

Variance:

Trust Variance.

Root Cause:

Consent boundary failure.

Resolution Candidate:

Strengthen Trust QA rejection rules for high-sensitivity memories without explicit consent.

## Non-Implementation Boundary

These examples do not define automated benchmark execution, scoring code, analytics, dashboards, notifications, integrations, or agents.
