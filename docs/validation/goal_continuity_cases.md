# Goal Continuity Cases v1

## Scenario G1: Goal Drift

- Scenario Name: Goal Drift
- Context: Goal active, calendar allocation near zero, recent decisions misaligned.
- Graph State: Goal active; supporting projects dormant; decisions affect Goal negatively.
- Relevant Metrics: Goal Drift high; Goal Attention Share low; Decision Stability weak.
- Expected Continuity Director Verdict: CONCERN
- Expected Awareness QA Outcome: SURFACE
- Expected Trust QA Outcome: ALLOW
- Expected Task Writer Artifact: Goal Drift Notice
- Expected Today Visibility: Visible
- Why This Outcome Is Correct: Evidence shows intention and behavior diverging; current decisions make it relevant now.

## Scenario G2: Competing Goal Pressure

- Scenario Name: Competing Goal Pressure
- Context: Two active goals depend on the same time window and both show current commitments.
- Graph State: Goal A active; Goal B active; shared commitments; calendar conflict evidence.
- Relevant Metrics: Goal Attention Share split; Commitment Risk moderate; Attention Load high.
- Expected Continuity Director Verdict: CONCERN
- Expected Awareness QA Outcome: SURFACE
- Expected Trust QA Outcome: ALLOW
- Expected Task Writer Artifact: Awareness Card
- Expected Today Visibility: Visible
- Why This Outcome Is Correct: Competing goals may distort continuity and decision quality; surfacing supports awareness without deciding priorities.

## Scenario G3: Old Goal With No Present Relevance

- Scenario Name: Old Goal With No Present Relevance
- Context: A prior aspiration exists in notes but no current project, decision, or commitment references it.
- Graph State: Goal latent; no active edges; old source exists.
- Relevant Metrics: Goal Alignment unknown; Awareness Yield low; Timing Fit absent.
- Expected Continuity Director Verdict: PASS
- Expected Awareness QA Outcome: SUPPRESS
- Expected Trust QA Outcome: ALLOW
- Expected Task Writer Artifact: None
- Expected Today Visibility: Not visible
- Why This Outcome Is Correct: Source exists, but relevance and why-now justification do not.
