# Project Continuity Cases v1

## Scenario P1: Project Momentum Loss

- Scenario Name: Project Momentum Loss
- Context: Active project with no meaningful activity for 21 days while linked goal remains active.
- Graph State: Project active; Goal active; Project supports Goal; owner present.
- Relevant Metrics: Momentum low; Activity Health weak; Goal Alignment high.
- Expected Continuity Director Verdict: CONCERN
- Expected Awareness QA Outcome: SURFACE
- Expected Trust QA Outcome: ALLOW
- Expected Task Writer Artifact: Project Momentum Alert
- Expected Today Visibility: Visible
- Why This Outcome Is Correct: Evidence shows a live project tied to an active goal may be losing continuity; low sensitivity and current context permit surfacing.

## Scenario P2: Hidden Abandonment

- Scenario Name: Hidden Abandonment
- Context: Project appears active through many small updates but no real progress or ownership movement.
- Graph State: Project active; commitments stale; decisions unresolved; artifacts repetitive.
- Relevant Metrics: Momentum artificially inflated; Context Stability low; Completion Progress weak.
- Expected Continuity Director Verdict: HIGH_CONCERN
- Expected Awareness QA Outcome: SURFACE
- Expected Trust QA Outcome: ALLOW
- Expected Task Writer Artifact: Project Momentum Alert
- Expected Today Visibility: Visible
- Why This Outcome Is Correct: False progress can hide abandonment; surfacing protects continuity and context quality.

## Scenario P3: Completed Project Still Appearing Active

- Scenario Name: Completed Project Still Appearing Active
- Context: Project outcome was delivered, but status remains active and creates attention noise.
- Graph State: Project completed evidence exists; status stale; related commitments fulfilled.
- Relevant Metrics: Completion Progress high; Attention Debt moderate; Context Stability strong.
- Expected Continuity Director Verdict: WATCH
- Expected Awareness QA Outcome: DEFER
- Expected Trust QA Outcome: ALLOW
- Expected Task Writer Artifact: Weekly Reflection Candidate
- Expected Today Visibility: Not visible today
- Why This Outcome Is Correct: The issue is administrative and not attention-worthy today; silence avoids awareness inflation.
