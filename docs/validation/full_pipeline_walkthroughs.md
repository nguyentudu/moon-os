# Full Pipeline Walkthroughs v1

## Purpose

This document walks five scenarios through the Constitutional Core.

Examples only. No implementation.

## Walkthrough 1: Project Momentum Loss

Graph:

Project active, Goal active, Project supports Goal, no meaningful activity for 21 days.

Metrics:

Momentum low, Activity Health weak, Goal Alignment high.

Continuity Director:

CONCERN because an active project tied to an active goal may be losing continuity.

Awareness QA:

SURFACE because the concern has current timing relevance and actionable awareness value.

Trust QA:

ALLOW because the project context is current and low sensitivity.

Task Writer:

Project Momentum Alert with evidence, why-now, confidence, and uncertainty.

Today Outcome:

Visible.

## Walkthrough 4: Attention Overload

Graph:

Fifteen valid concerns exist; two have high commitment or project impact.

Metrics:

Attention Load high, Signal Density low, Awareness Yield concentrated.

Continuity Director:

CONCERN because many continuity signals exist.

Awareness QA:

PRIORITIZE top two and suppress or defer most others.

Trust QA:

ALLOW only permitted low-sensitivity items.

Task Writer:

Creates artifacts only for approved high-value items.

Today Outcome:

Only a small number of high-value items visible.
