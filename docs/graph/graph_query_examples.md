# Graph Query Examples v1

## Purpose

This file lists conceptual query examples that future graph, metrics, or reasoning layers may support.

These are examples only. They are not implementation requirements, database queries, retrieval instructions, or product features.

## Examples

Which commitments are linked to active goals?

Which projects have no activity for 30 days?

Which people are connected to high-risk commitments?

Which decisions affect Goal Alpha?

Which learning topics have become dormant?

Why is this insight being surfaced today?

Which projects support goals that are drifting?

Which commitments have evidence of follow-up but no closure evidence?

Which relationships are active but have boundary-sensitive evidence?

Which documents support decisions that were later reversed?

Which memories reference both a person and a project?

Which events affected relationship state?

Which threads contain memories with weak confidence?

Which insights are derived from direct evidence?

Which graph objects should remain silent because attention cost exceeds awareness value?

## Query Governance

Every future query result should be able to expose:

- source
- evidence
- confidence
- relevance
- why now
- consent boundary

## Non-Implementation Boundary

This file does not define query syntax, indexing, storage, retrieval, ranking, or reasoning behavior.
