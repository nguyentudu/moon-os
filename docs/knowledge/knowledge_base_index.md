# Continuity Knowledge Base Index v1

## Purpose

This index defines the first knowledge layer for Moon OS. The knowledge layer names the domains Moon must understand before any graph, metrics, reasoning, or Today experience can be designed.

This layer is not an implementation plan. It is a reusable conceptual foundation for continuity across projects, goals, commitments, relationships, decisions, learning, attention, and trust.

Future graph work may reference these files to define node and edge meaning.

Future metrics work may reference these files to define candidate scoring dimensions.

Future reasoning work may reference these files to justify surfacing, ranking, or silence.

Future Today work may reference these files to decide what deserves attention.

## Hard Boundary

This knowledge base does not define product features, database schemas, user interface behavior, reminders, agents, or memory implementation.
