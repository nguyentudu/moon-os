# Runtime Demonstration Overview v1

## Purpose

The Moon Runtime Demonstration exists to show reviewers how Moon turns structured continuity state into a Today awareness item.

It is a glass box demo:

Evidence -> Graph -> Metrics -> Reasoning -> Trust -> Today Artifact

## What It Proves

The demo proves that Moon Today can show why an awareness item appeared.

It demonstrates that each surfaced item can be traced through:

- input state
- continuity graph facts
- metric signals
- constitutional reasoning
- trust boundary review
- final Today artifact

## What It Does Not Prove

This demo does not prove real-world ingestion.

It does not prove calendar, email, chat, document, or external data connection.

It does not prove automation, notifications, reminders, agents, auth, or database storage.

It does not claim that the synthetic cases are real user data.

## Why No Real Data Is Used Yet

Moon has not entered real data ingestion.

Using synthetic canonical scenarios keeps the review focused on whether the reasoning path is understandable and trustworthy before any real personal data is introduced.

This protects trust, attention, and consent boundaries while still allowing reviewers to inspect the runtime shape.

## Reviewer Question

After viewing the demo, reviewers should be able to answer:

Can I see why Moon showed this?
