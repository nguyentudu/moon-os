# Experience Redesign Notes v1

## What Changed

The Moon Today prototype was reframed from an awareness artifact gallery into a personal continuity experience.

The first screen now explains Moon as a place that helps the user return to what is still open without having to remember everything.

The 5 existing awareness artifacts are grouped under three human-facing states:

- Quay lại
- Để yên
- Suy ngẫm

## Why Changed

The previous prototype made reviewers evaluate artifact structure first.

The redesign is intended to help a first-time reviewer understand within 30 seconds that Moon is about personal continuity, not AI-generated advice, task management, or reminder pressure.

The target feeling is:

"Moon helps me return to what is still open."

## What Was Preserved

The existing 5 awareness artifacts were preserved.

No artifact meaning was changed.

Each item still shows:

- Title
- Summary
- Why It Matters
- Why Now
- Evidence
- Confidence
- Trust Notes
- Suggested Reflection

No new reasoning, fake user data, ingestion, auth, agents, notifications, or database behavior was added.

## What Reviewers Should Evaluate Now

Reviewers should evaluate:

- whether the first screen explains Moon quickly
- whether the experience feels like morning awareness
- whether "Quay lại / Để yên / Suy ngẫm" makes the items easier to understand
- whether the tone feels trustworthy
- whether the prototype feels like continuity support rather than advice
- whether they would want to return to it in the morning

Reviewers should not evaluate internal architecture.
