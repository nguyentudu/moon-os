# Reviewer Questionnaire v1

## Instructions

Open the prototype first:

https://moon-today-prototype-production.up.railway.app

Spend 3-5 minutes.

Review it as if this is something you see in the morning.

Do not evaluate Moon's architecture or implementation.

## Questions

1. What do you think this product is trying to do?
2. Which item felt most useful?
3. Which item felt least useful?
4. Did anything feel intrusive, judgmental, or uncomfortable?
5. Did you understand "why now" for each item?
6. Did the evidence feel enough?
7. Did the tone feel trustworthy?
8. Would you want to see this in the morning?
9. Would you want more, less, or the same amount?
10. Would you open this again tomorrow?
11. What should be removed?
12. What should be improved?

## Rating Fields

Rate each from 1 to 5.

- Clarity
- Relevance
- Trust
- Usefulness
- Emotional safety
- Desire to reopen
