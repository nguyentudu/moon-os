# TTC Lab Participant Instructions v1

## Purpose

These instructions are for testers using Moon Context Challenge.

## Test Modes

### Manual Mode

Use the Moon repo to find answers manually.

Decision Memory answers are hidden.

### Decision Memory Mode

Use the structured Decision Memory answer shown in the lab.

Rewrite the answer in your own words.

## Steps

1. Open the TTC Lab URL.
2. Choose your assigned mode.
3. For each question, click Start timer.
4. Read using the allowed material for your mode.
5. Write your answer.
6. Select confidence 1-5.
7. Click Stop timer.
8. Copy the result.
9. Send copied results back manually.

## Rules

- Do not switch modes mid-test unless instructed.
- Do not use Decision Memory answers in Manual Mode.
- Do not invent evidence.
- Mark uncertainty clearly.
- Send results manually; the page does not submit anything.
