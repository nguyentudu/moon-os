# TTC Lab Scoring Notes v1

## Purpose

Define how copied TTC Lab responses should be scored after real tester evidence exists.

## Measures

### Time To Answer

Use the copied elapsed time per question.

### Completeness

Score manually from 1 to 5:

1. Unsupported or incorrect.
2. Partial answer with little evidence.
3. Reasonable answer with some evidence.
4. Strong answer with relevant evidence.
5. Complete answer with evidence, validity, and uncertainty.

### Confidence

Use participant self-rating from 1 to 5.

### Perceived Difficulty

Collect separately in the reviewer form or manual follow-up.

## Comparison

Compare Manual Mode and Decision Memory Mode on:

- median time
- average completeness
- average confidence
- reported difficulty
- evidence quality

## No Fake Result Rule

Do not score or summarize results until real copied tester responses exist.
