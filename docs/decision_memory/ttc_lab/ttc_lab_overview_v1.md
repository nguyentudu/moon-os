# TTC Lab Overview v1

## Purpose

Moon TTC Lab is a tiny public measurement UI for Time To Context testing.

It lets real testers compare:

- manual repo/context reading
- Decision Memory assisted understanding

## What It Measures

The lab captures evidence for:

- time to answer
- answer completeness
- participant confidence
- perceived difficulty

## What It Does Not Measure Yet

The lab does not automatically submit or aggregate results.

It does not prove Time To Context reduction until real tester evidence is collected.

## Public Location

The Railway-served lab is intended to be available at:

https://moon-today-prototype-production.up.railway.app/ttc/

## Boundary

This is not a product UI.

No backend, auth, database, ingestion, agent, chat, notification, or fake result system is included.
