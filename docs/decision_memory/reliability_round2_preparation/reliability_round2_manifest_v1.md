# Reliability Round 2 Manifest v1

## Purpose

Prepare Reliability Round 2 without changing the Decision Definition.

Round 2 is intended to improve reviewer ability to distinguish:

- Decision
- Not Decision
- Unclear

## Source Evidence

This preparation uses:

- docs/decision_memory/reliability_results/
- docs/decision_memory/boundary_audit/
- docs/decision_memory/extraction_validation/
- docs/decision_memory/extraction_reliability/
- docs/decision_memory/reliability_lab/

## Problem Being Addressed

Reliability Round 1 produced no Not Decision classifications.

Boundary Audit found likely combined causes:

- UI / instruction framing made Decision more salient than Not Decision.
- expected Not Decision excerpts included ambiguous decision-adjacent examples.
- reviewers used Unclear as the non-Decision escape hatch.

## Boundaries

This preparation does not:

- modify the Decision Definition
- modify decision records
- create new excerpts
- create fake reviewer data
- calculate new agreement
- run reviewer studies
- create reliability claims
- expand Decision Memory

## Round 2 Readiness Goal

Round 2 should test whether clearer training examples and instructions make Not Decision usable for reviewers.
