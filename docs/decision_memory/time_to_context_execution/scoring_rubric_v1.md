# Scoring Rubric v1

## Purpose

Define how Time To Context benchmark responses should be scored.

No scores are recorded here.

## Measures

### Time To Answer

Measure elapsed time from question start to first credible answer.

Record as minutes and seconds.

### Answer Completeness

Score 1-5:

1. Incorrect or unsupported.
2. Partial answer with weak support.
3. Reasonable answer with some evidence.
4. Strong answer with relevant evidence.
5. Complete answer with decision/evidence, validity, and uncertainty.

### Confidence

Participant self-rating 1-5:

1. Not confident.
2. Slightly confident.
3. Moderately confident.
4. Confident.
5. Very confident.

### Perceived Difficulty

Participant self-rating 1-5:

1. Very easy.
2. Easy.
3. Moderate.
4. Hard.
5. Very hard.

## Group Comparison

Compare Group A and Group B on:

- median time to answer
- average completeness
- average confidence
- average perceived difficulty
- evidence quality

## Hard Rules

- Do not score missing responses.
- Do not infer participant timing.
- Do not invent confidence.
- Do not claim time saved unless measured.
- Record NO_CLAIM if no participant evidence exists.
