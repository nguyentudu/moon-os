# Reliability Lab Reviewer Instructions v1

## Purpose

Guide reviewers through the Moon Reliability Lab.

## Instructions

1. Open the Reliability Lab URL.
2. Choose English or Tiếng Việt.
3. Enter your reviewer ID and date.
4. Read each excerpt.
5. Classify it as Decision, Not Decision, or Unclear.
6. Add an optional comment if your classification feels difficult.
7. Use Previous and Next to move through all 50 excerpts.
8. Click Copy all results.
9. Send the copied text back manually.

## Important

Do not use AI assistance while classifying.

There are no right answers shown in the UI.

The study is testing whether the definition is clear enough for humans to apply.
