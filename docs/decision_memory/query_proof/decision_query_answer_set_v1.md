## Q1: Why did Moon build Constitutional Core before UI?

Retrieved decision record(s):

- DM-001
- DM-009

Reason:

DM-001 records the sequencing choice to define law, knowledge, graph, metrics, and reasoning before Today UI. DM-009 explains why Moon is constitutional continuity infrastructure rather than a generic prompt wrapper.

Evidence links:

- PERSONAL_CONTINUITY_MANIFEST.md
- TRUST_GOVERNANCE.md
- ATTENTION_CONSTITUTION.md
- docs/knowledge/knowledge_base_index.md
- docs/graph/continuity_graph_schema_v1.md

- docs/reasoning/continuity_director_charter_v1.md
- docs/reasoning/awareness/awareness_qa_charter_v1.md
- docs/reasoning/trust/trust_qa_charter_v1.md

Current validity:

- DM-001: valid
- DM-009: valid

Concise answer:

Moon built the Constitutional Core before UI because the product depends on evidence, consent, attention governance, graph structure, metrics, and reasoning boundaries. A UI first approach would risk showing awareness without enough source, trust, or interruption discipline.

Confidence:

high

What is still unknown:

No measured Time To Context test has yet shown how much faster this answer is than reading the full repo manually.
