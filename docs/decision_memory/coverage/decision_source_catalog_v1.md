# Decision Source Catalog v1

## Purpose

Catalog major Moon sources that contain or imply historical product, architecture, trust, validation, runtime, and measurement decisions.

This catalog is not a completeness claim.

## Source Categories

| Source Category | Representative Files | Coverage Use |
| --- | --- | --- |
| Manifest | PERSONAL_CONTINUITY_MANIFEST.md | Foundational purpose and evidence-first decisions |
| Trust Governance | TRUST_GOVERNANCE.md, docs/reasoning/trust/* | Consent, recall, boundary, and Trust QA decisions |
| Attention Constitution | ATTENTION_CONSTITUTION.md, docs/reasoning/awareness/* | Attention, interruption, why-now, and Awareness QA decisions |
| Runtime Demo | apps/moon_today_experience_prototype/runtime.html, docs/runtime_demo/* | Glass box reasoning and synthetic scenario decisions |
| Human Evaluation | docs/evaluation/human/* | Human evidence, reviewer packet, campaign, and NO_CLAIM decisions |
| TTC Lab | apps/moon_ttc_lab/*, docs/decision_memory/ttc_lab/* | Time To Context measurement UI decisions |
| Benchmark Documents | docs/validation/*, docs/decision_memory/time_to_context_execution/* | Constitutional benchmark, audit, and TTC benchmark decisions |
| Deployment Reviews | docs/deployment/* | Public prototype, runtime, and TTC Lab deploy decisions |
| Architecture Notes | README.md, docs/knowledge/*, docs/graph/*, docs/metrics/*, docs/reasoning/*, docs/today/* | Layering, ontology, metrics, reasoning, and Today contract decisions |
| Review Logs | *_REVIEW.md files across docs and apps | Gate results, proof artifacts, verdict decisions |

## Source Boundary

Only repository artifacts are cataloged.

No private memory, external chat, email, calendar, or non-repo source is included.
