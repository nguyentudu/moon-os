# Disagreement Analysis Template v1

## Purpose

Provide a template for analyzing disagreements after reviewer responses exist.

No disagreements are recorded yet.

## Disagreement Categories

### Decision vs Not Decision

Use when at least one reviewer classified an excerpt as decision and another classified it as not decision.

### Decision vs Unclear

Use when reviewers disagree between decision and unclear.

### Not Decision vs Unclear

Use when reviewers disagree between not decision and unclear.

### Three-Way Split

Use when decision, not decision, and unclear all appear for the same excerpt.
