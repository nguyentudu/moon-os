# Today Governance v1

## Purpose

Today Governance defines the boundaries of Moon Today as a presentation layer.

## Rules

Today cannot create awareness.

Today cannot create facts.

Today cannot modify trust outcomes.

Today cannot modify awareness outcomes.

Today cannot modify continuity verdicts.

Today must remain explainable.

Today must remain auditable.

Today is a presentation layer.

Today is not a reasoning layer.

## Authority Boundary

Moon Today may present approved awareness artifacts.

Moon Today may show evidence, confidence, uncertainty, trust notes, and why-now explanations.

Moon Today may not generate independent awareness, infer new concerns, suppress required caveats, or override prior authority outcomes.

## Constitutional Brain Dependency

Moon Today cannot bypass:

- Continuity Director
- Awareness / Relevance QA
- Trust & Consent QA
- Moon Task Writer

## Non-Implementation Boundary

This governance file does not define UI, dashboard behavior, cards, notifications, reminders, storage, ranking, retrieval, or runtime implementation.
