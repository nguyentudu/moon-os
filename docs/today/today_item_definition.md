# Today Item Definition v1

## Purpose

This document defines what constitutes a valid Moon Today item.

A Today item is an approved awareness artifact that has passed the Constitutional Brain and is eligible for presentation by a future Today Runtime.

## Required Fields

A valid Today item must contain:

- Title
- Summary
- Why It Matters
- Why Now
- Evidence
- Confidence
- Uncertainty
- Trust Notes, if applicable
- Source References

Without these fields, the item is invalid.

## Field Meaning

### Title

A concise name for the awareness item.

### Summary

A plain-language summary of the concern or awareness artifact.

### Why It Matters

The continuity reason this item deserves awareness.

### Why Now

The timing reason this item deserves awareness today.

### Evidence

The source-linked basis for the item.

### Confidence

The stated confidence or confidence caveat.

### Uncertainty

Known ambiguity, missing context, or limits.

### Trust Notes

Consent, sensitivity, or trust caveats required by Trust QA.

### Source References

References to the sources supporting the artifact.

## Invalid Item Conditions

An item is invalid if it lacks evidence, source references, why-now explanation, confidence, or required trust notes.

## Non-Implementation Boundary

This definition does not define UI cards, data schemas, storage, rendering, or runtime validation logic.
