# Today Priority Model v1

## Purpose

This model defines conceptual categories for Moon Today items.

Categories only. No ranking algorithm.

## Categories

### Priority Awareness

High-value approved awareness item that should compete for top attention.

Example: critical commitment risk with strong why-now timing.

### Standard Awareness

Approved awareness item with clear evidence, timing, and trust permission.

Example: active project concern with moderate continuity impact.

### Reflection Candidate

Approved item better suited for reflection than immediate action.

Example: long-form pattern around goal drift or learning continuity.

### Relationship Watch

Approved relationship continuity concern.

Example: permitted relationship watch with visible trust notes.

### Commitment Risk

Approved commitment continuity concern.

Example: current responsibility with near deadline and clear consent.

### Goal Drift

Approved goal alignment concern.

Example: active goal showing divergence between intention and effort.

### Project Momentum

Approved project continuity concern.

Example: project linked to active goal with momentum loss.

### Learning Continuity

Approved learning concern.

Example: important learning thread showing fragmentation.

## Non-Implementation Boundary

This model does not define ranking, sorting, scoring, layout, cards, notifications, reminders, or runtime prioritization.
