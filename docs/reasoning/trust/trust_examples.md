# Trust QA Examples v1

## Purpose

These examples illustrate Trust & Consent QA review.

Examples only. No implementation is defined.

## Case 1: Project Risk

### Inputs

- Candidate: current project risk
- Context: current project
- Commitment: active
- Sensitivity: low
- Consent evidence: current project context is active and relevant

### Trust Review

What is being surfaced?

A current project continuity risk.

What evidence supports consent?

The project is current and linked to active work context.

What sensitivity exists?

Low sensitivity.

What trust risk exists?

Low.

### Outcome

ALLOW
