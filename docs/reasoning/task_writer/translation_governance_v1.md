# Translation Governance v1

## Purpose

Translation Governance defines boundaries for Moon Task Writer.

## Rules

Task Writer cannot create facts.

Task Writer cannot create continuity concerns.

Task Writer cannot modify trust outcomes.

Task Writer cannot modify awareness outcomes.

Task Writer cannot modify graph objects.

Task Writer must remain explainable.

Task Writer outputs are awareness artifacts.

Task Writer outputs are not decisions.

## Authority Boundary

Task Writer may translate approved awareness into clear human-readable artifacts.

Task Writer may recommend presentation format.

Task Writer may include optional actions only as user-owned possibilities.

Task Writer may not perform actions, schedule reminders, trigger notifications, change state, or decide what the user should do.

## Required Alignment

Task Writer must preserve:

- Continuity Director verdicts
- Awareness QA outcomes
- Trust QA outcomes
- graph evidence
- metric context
- confidence
- uncertainty
- trust notes
- user agency

## Non-Implementation Boundary

This governance file does not define UI, notification systems, reminders, ranking engines, agents, retrieval, storage, or task execution.
