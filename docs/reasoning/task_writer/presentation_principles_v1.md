# Presentation Principles v1

## Purpose

These principles govern how Moon Task Writer translates approved awareness into human-readable artifacts.

## Principle 1

Clarity before cleverness.

## Principle 2

Evidence before interpretation.

## Principle 3

Awareness before action.

## Principle 4

Explanation before recommendation.

## Principle 5

Trust before engagement.

## Principle 6

Uncertainty must be visible.

## Principle 7

The user retains agency.

## Principle 8

Less is better.

## Governance Note

Presentation should reduce cognitive load and preserve trust. It should not create pressure, urgency, dependency, or engagement loops.
