# Human Review Framework v1

## Purpose

Evaluate whether Moon Today awareness artifacts are understandable, relevant, trustworthy, and useful to a human reviewer.

## Review Questions

Can the user understand this?

Does it feel relevant?

Does it feel trustworthy?

Does it explain why now?

Does it provide evidence?

Does it respect agency?

Does it avoid manipulation?

Would you want to see this?

Would you want less of this?

Would you trust this tomorrow?

## Review Guidance

Human review should judge the artifact as an experience, not only as a technically correct output.

A technically correct awareness item that feels confusing, irrelevant, or intrusive should be treated as a failure signal.

## Non-Implementation Boundary

This framework does not define surveys, analytics, UI, notifications, reminders, ingestion, retrieval, agents, or external integrations.
