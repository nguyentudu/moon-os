# Today Screen Prototype v1

## Purpose

Define a simple human-reviewable Today screen structure.

No UI framework.

No design system.

No visual design specification.

Only presentation structure.

## Required Sections

### Today's Awareness

Short heading for the day's approved awareness.

### Awareness Items

Human-readable awareness artifacts selected for review.

### Evidence

Source-linked support for each item.

### Confidence

Confidence level or caveat for each item.

### Trust Notes

Consent, sensitivity, or caution notes where applicable.

### Reflection Prompt

Optional non-directive reflection prompt that preserves user agency.

## Prototype Flow

Today's Awareness

↓

Awareness Items

↓

Evidence

↓

Confidence

↓

Trust Notes

↓

Reflection Prompt

## Non-Implementation Boundary

This prototype does not define UI layout, styling, components, dashboard behavior, notifications, reminders, ingestion, retrieval, or agents.
