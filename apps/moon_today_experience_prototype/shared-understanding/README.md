# Moon Shared Understanding Lab

Static research UI for collecting raw Shared Understanding Study answers.

## Scope

The app collects:

- participant_id
- language
- case_id
- recommendation
- rationale
- evidence_used
- risks_tradeoffs
- uncertainty
- confidence
- start_time
- end_time
- elapsed_seconds
- created_at

## Boundaries

This app does not:

- score answers
- rank participants
- claim shared understanding
- claim Team Continuity
- claim economic value
- submit to a backend
- use a database
- create fake participant data

Participants must copy or download raw JSON for manual import.
