# Moon TTC Study Lab v2

Research UI for Time To Context study execution.

## Purpose

Collect raw participant evidence comparing:

- Manual Context Discovery
- Moon Decision Memory Assisted Context Discovery

## Boundaries

This app does not:

- score answers
- aggregate results
- claim improvement
- claim economic value
- create fake participants
- submit to a backend

## Data Capture

The app records in browser memory:

- participant_id
- mode
- question_id
- question
- answer
- start_time
- end_time
- elapsed_seconds
- created_at

Participants copy or download raw JSON for manual import.
