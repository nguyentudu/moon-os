# Today Runtime Flow v1

## Purpose

This document defines the conceptual runtime flow for Moon Today Runtime MVP.

Examples only.

No implementation.

## Flow

Approved Awareness Artifact

↓

Surface Contract Validation

↓

Valid Today Item

↓

Render

↓

User Sees Awareness

## Flow Notes

The artifact must already be approved by:

- Continuity Director
- Awareness QA
- Trust QA
- Task Writer

The runtime does not perform reasoning.

The runtime does not retrieve memory.

The runtime does not create reminders.

The runtime does not send notifications.

The runtime does not connect external systems.

## Non-Implementation Boundary

This flow does not define code, services, storage, UI, scheduling, notifications, reminders, agents, or integrations.
