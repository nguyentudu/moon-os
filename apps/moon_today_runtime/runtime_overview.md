# Runtime Overview v1

## Purpose

Moon Today Runtime MVP demonstrates the narrow runtime path from approved awareness artifact to rendered Today item.

This is the first runtime layer of Moon OS.

TODAY RUNTIME MVP

## Runtime Scope

The runtime accepts one approved awareness artifact and produces one rendered Today item.

No external systems are used.

No database is used.

No retrieval is performed.

No notifications are created.

No reminders are created.

No agents are executed.

No user accounts or integrations are required.

## Runtime Boundary

The runtime does not decide significance, attention priority, consent, trust, or presentation governance.

It only demonstrates that a Task Writer artifact satisfying the Today Surface Contract can become a visible awareness structure.
