# Today Item Renderer v1

## Purpose

The Today Item Renderer defines the presentation structure for a rendered Today item.

No styling.

No UI framework.

No design system.

Only presentation structure.

## Rendered Today Item Sections

### Title

The artifact title.

### Summary

The artifact summary.

### Why It Matters

The continuity significance.

### Why Now

The timing justification.

### Evidence

The source-linked support.

### Confidence

The confidence statement.

### Uncertainty

The uncertainty or caveat.

### Trust Notes

The trust context, if applicable.

### Source References

The source references supporting the artifact.

## Rendering Boundary

Rendering may only present approved artifact content.

Rendering may not invent facts, change verdicts, hide uncertainty, remove trust notes, or create actions.

## Non-Implementation Boundary

This file does not define UI components, visual styling, layout, notifications, reminders, or runtime code.
